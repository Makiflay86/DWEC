Ejercicio 11

Caracteres Especiales Distingue diferentes tipos de caracteres. Por ejemplo, distinguir entre letras y dígitos, cualquier caracter o saltos de linea.
Aserciones Las aserciones incluyen límites, que indican el comienzo y el final de líneas y palabras, y otros patrones que indican de alguna manera que el reconocimiento es posible (incluidas las expresiones anticipadas, inversas y condicionales).
Grupos y rangos Indica grupos y rangos de caracteres de expresión. Los Grupos con paréntesis se utilizan como dispositivos de memoria. La coincidencia realizada con esta parte del patrón se recuerda para su uso posterior.
Cuantificadores Indica el número de caracteres o expresiones que deben coincidir.
Escapes de propiedades Unicode Distinguen según las propiedades de los caracteres Unicode, por ejemplo, letras mayúsculas y minúsculas, símbolos matemáticos y de puntuaciónsafsad.

Como son las dos formas de ejecutar una expresion regular?

Ejercicio 15

Método Devuelve Descripción
RegExp.exec(String) Array Ejecuta una búsqueda por una coincidencia en una cadena. Devuelve un arreglo de información o null en una discrepancia.
RegExp.test(String) Booleano Prueba una coincidencia en una cadena. Devuelve true o false.
String.match(RegExp) Array Devuelve un arreglo que contiene todas las coincidencias, incluidos los grupos de captura, o null si no se encuentra ninguna coincidencia.
String.matchAll(RegExp) Array Devuelve un iterador que contiene todas las coincidencias, incluidos los grupos de captura.
String.search(RegExp) numero (índice) Prueba una coincidencia en una cadena. Devuelve el índice de la coincidencia, o -1 si la búsqueda falla.
String.replace(RegExp) String Ejecuta una búsqueda por una coincidencia en una cadena y reemplaza la subcadena coincidente con una subcadena de reemplazo.
String.replaceAll(RegExp) String Ejecuta una búsqueda de todas las coincidencias en una cadena y reemplaza las subcadenas coincidentes con una subcadena de reemplazo.
String.split(RegExp) Array Utiliza una expresión regular o una cadena fija para dividir una cadena en un arreglo de subcadenas.
