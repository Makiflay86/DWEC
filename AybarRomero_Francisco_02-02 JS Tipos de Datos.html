3º ¿Qué etiqueta se utiliza para insertar código JavaScript en HTML?

<script><script>

4º ¿Qué ventajas tiene incrustar el código JavaScript en un archivo externo?

Mejora la organización del código, facilita el mantenimiento, permite la reutilización de funciones en diferentes páginas y acelera la carga de la web al aprovechar el caché del navegador.

5º ¿Qué extensión suelen tener los archivos que contienen código JavaScript?

.js -> javascript

6º ¿Cómo se referencia un archivo .js en HTML?

Habría que añadir dentro de la etiqueta script esto: src="./script.js"

7º ¿Qué debe hacer si el archivo JavaScript se encuentra en otro directorio?

Se puede hacer mediante dos formas, usando rutas absolutas o rutas relativas. Lo mejor es usar las rutas relativas ya que no tendras que modificar esta ruta si mueves el directorio de trabajo.

8º ¿Qué etiqueta se usa para mostrar un mensaje si el navegador no soporta JavaScript?

La etiqueta <noscript>.

9º ¿Qué diferencia hay al insertar JavaScript en XHTML?

Si te equivos lo más mínimo XHTML no te dirá donde esta el error, te pondrá que todo esta mal.

10º ¿Qué es la ofuscación de código y por qué se utiliza?

Es modificar el programa para hacer que su código sea mucho más difícil de entender para los humanos. Normalmente se suele usar para la seguridad, ocultar código malicioso, reducir el tamaño del archivo.

11º ¿Cómo se llama el lenguaje que se menciona en el texto?

Pues no se que texto.

12º ¿Qué nombre le dio Microsoft a JavaScript?

JScript.

13º ¿Por qué Microsoft usó el nombre Jscript?

Como forma de competir con Netscape, el creador de JavaScript.

14º ¿Qué problema surgió con la versión de JScript de Microsoft?

Problemas de compatibilidad en el desarrollo web.

15º ¿Qué es el DOM y por qué fue diseñado?

Interfaz de programación para documentos HTML y XML que representa la estructura de la página como un árbol de nodos. Fue diseñado para la interactividad y como estándar de acceso.

16º ¿Qué navegadores incorporaron el DOM desde su primera versión?

Netscape navegador, Internet Explorer y Mozilla.

17º ¿Qué estándar regula el lenguaje JavaScript desde 1997?

ECMAScript.

18º ¿Qué documento detalla las especificaciones del estándar ECMA?

Define como es el lenguaje como JavaScript.

19º ¿Qué relación tiene JavaScript con Java?

Ambos son lenguajes de programación orientados a objetos y que su sintaxis se inspira en C.

20º ¿Quién desarrolló originalmente JavaScript y con qué nombre?

Brendan Eich desarrolló JavaScript en 1995 para Netscape, inicialmente con el nombre de Mocha.
