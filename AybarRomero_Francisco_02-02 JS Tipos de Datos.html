3º ¿Qué etiqueta se utiliza para insertar código JavaScript en HTML?

<script><script>

4º ¿Qué ventajas tiene incrustar el código JavaScript en un archivo externo?

Mejora la organización del código, facilita el mantenimiento, permite la reutilización de funciones en diferentes páginas y acelera la carga de la web al aprovechar el caché del navegador.

5º ¿Qué extensión suelen tener los archivos que contienen código JavaScript?

.js -> javascript

6º ¿Cómo se referencia un archivo .js en HTML?

Habría que añadir dentro de la etiqueta script esto: src="./script.js"
